# Moluoxixi AI Workflow Distribution

Moluoxixi AI Workflow Distribution is a superpowers-based, skills-first workflow distribution for personal AI coding.

It is built on top of [superpowers](https://github.com/obra/superpowers) and packages a consistent workflow layer across Claude, Codex, Qoder, and openCode.

## What This Distribution Includes

- First-party skills in `skills/` for local workflow defaults and stack guidance
- Vendor skills managed through `manifests/` and projected into each host
- First-party agents in `agents/` for orchestration roles
- Multi-host install/upgrade docs in `.claude/`, `.codex/`, `.qoder/`, and `.opencode/`

## Host Support

- Claude / Claude Code
- Codex CLI
- Qoder IDE
- openCode
