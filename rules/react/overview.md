# React Rules Overview

适用于 React / Next.js 项目。

## 架构原则

- 组件按职责拆分
- 优先保证数据流与状态边界清晰
- 保持客户端与服务端职责分离
- 页面编排结合 `frontend-design` 与 `cache-components` 等外部 skill
