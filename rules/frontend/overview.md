# Frontend Rules Overview

关注页面结构、组件边界、状态管理与视觉一致性。

## 架构原则

- 页面和组件职责清晰
- 首屏、交互和错误态完整
- 样式策略一致，不混乱叠加
- 与 UI 测试规则一起使用
